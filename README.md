# Quran Video Editor

تطبيق سطح مكتب احترافي لتحرير فيديوهات القرآن الكريم باستخدام Electron + React + TypeScript مع دعم أدوات تحرير متقدمة وتقنيات الذكاء الاصطناعي.

## المتطلبات
- Node.js 18+
- FFmpeg مثبت على النظام
- مفتاح OpenAI محفوظ في متغير البيئة `OPENAI_API_KEY`

## التشغيل محليًا
```bash
npm install
npm run dev
```

## بناء نسخة الإنتاج
```bash
npm run build
npm run dist
```

## المميزات الأساسية
- رفع الفيديو ومعاينته وعرض بياناته.
- التعرف التلقائي على الكلمات عبر Whisper API.
- 30 أداة تحرير (نصوص، فيديو، صوت) مع واجهة تحكم.
- شريط زمني للكلمات المستخرجة.
- خيارات تصدير متعددة مع التحكم في الجودة ومعدل الإطارات.

## هيكل المشروع
- `src/main`: عمليات Electron الرئيسية ومعالجة الفيديو.
- `src/renderer`: واجهة المستخدم React.
- `src/services`: خدمات مساعدة للتكامل.
- `src/types`: تعريفات TypeScript.

## ملاحظات
- يجب توفير FFmpeg في PATH.
- المسار الافتراضي للتصدير يمكن تغييره من واجهة التطبيق.
